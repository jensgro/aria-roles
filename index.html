ARIA Role Categories

Github Page

When WAI ARIA 1.0 became a W3C Recommendation in March 2014, there were four categories for roles.

  1. Abstract Roles
  2. Widget Roles and Widget Containers
  3. Document Structure Roles
  4. Landmark Roles

When WAI ARIA 1.1 became a W3C Recommendation in 14 December 2017, some additional categories were introduced and the categories then became:

  1. Abstract Roles
  2. Widget Roles and Widget Containers
  3. Document Structure Roles
  4. Landmark Roles
  5. Live Region Roles
  6. Window Roles

ARIA 1.2 became a Working Draft in July 2018 and these categories have remained the same.

With the introduction of these new categories, some existing 1.0 roles were moved to a new category.

The chart below shows all WAI ARIA roles including notes on when they changed category (c), or where newly introduced (n).
